# Frontend

Next.js web application for AutoTreasury AI.

## Structure

- `components/` - React components
- `pages/` - Next.js pages
- `lib/` - Utility functions and helpers
- `hooks/` - Custom React hooks

## Setup

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

# Contracts

Smart contracts for AutoTreasury AI on BNB Chain.

## Structure

- `core/` - Core treasury management contracts
- `interfaces/` - Interface definitions for protocols (PancakeSwap, Venus, etc.)
- `libraries/` - Shared utility libraries
- `mocks/` - Mock contracts for testing

## Setup

```bash
pnpm install
pnpm compile
pnpm test
```
